Road Map

Proses klasifikasi sentiment.

  • Data

    To obtaining data

    A major part of machine learning or deep learning is Data! In fact, it's so important that the accuracy we will get depens on our data, the better our data then the better result we will get. Data can be obtained from open sources or we can scrape from certain websites or from other databases.

  • Preprocessing

    To Cleaning the data

    Usually the dataset that we have obtained needs to be cleaned. Especially when dataset is raw and unstructural. In our case the dataset is typical text data such words. When we are going to classify sentiment in the text type of data, there are many steps we need to do with the dataset before we feed it into our machine. Since the words in the dataset does not always correct in spelling we need to Normalize it. And also we need to remove all the stopwords and punctuantions from our dataset.

  • Training

    To train the machine

    Then, we are ready to feed the data to our machine. Amount of data we are going to feed into Training process usually much larger than data for Testing. We will use 80:20 to divide our dataset into Data Training and Data Testing. In our machine we use Deep Learning approach. Using FastText Classifier our machine will learn from our data and will generate a model as a result.

  • Testing

    Evaluate the model

    This time we will test our model. In this part we will see how good is our model to predict the sentiment of our data Testing. This process will return the accuracy of the prediction our machine have made.



  • That's it!
